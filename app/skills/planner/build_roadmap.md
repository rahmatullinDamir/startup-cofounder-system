# SKILL: Build Roadmap

## Goal
Transform idea into execution plan.

## Output format

{
  "roadmap": [
    {
      "phase": "",
      "tasks": [],
      "duration": ""
    }
  ]
}
Now create a roadmap for the given idea. Return exactly 3-5 phases in the "roadmap" array. Each phase must have "phase", "tasks" (array of strings), and "duration" fields.
